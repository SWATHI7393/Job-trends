# **App Name**: JobTrend Pro

## Core Features:

- Job Trend Prediction: Predict future job market trends based on user inputs such as job role, location, experience, and skills, using a predictive model that acts as a tool for providing forecasts and insights.
- Skill Gap Analysis: Analyze skill gaps based on job roles and generate recommendations for skills to learn.
- Resume Analysis: Analyze uploaded resumes to extract skills, calculate job match scores, and suggest ATS improvements.
- Market Insights Dashboard: Display interactive charts and cards presenting trending job roles, skill demand heatmaps, and other market insights.
- User Authentication: Implement user authentication with Firebase Auth, including login and signup functionalities.
- Data Persistence: Enable saving predictions and user profiles in Firestore.
- UI Components: Provide UI components like tag creation, meter, missing skills as chips.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calm and modern feel.
- Background color: Very light lavender (#F5F5FF) - almost white - to ensure contrast while remaining in the pastel color scheme.
- Accent color: Pale purple (#D8BFD8) for interactive elements like buttons and tags.
- Body and headline font: 'Inter', a sans-serif font for a modern, clean interface. It's suitable for headlines or body text.
- Use simple, outline-style icons to maintain a minimal and clean aesthetic.
- Implement a floating left sidebar for navigation, a spacious workspace, and card-based layouts for content display, similar to Canva's interface.
- Incorporate subtle hover animations and smooth transitions for an engaging user experience.
